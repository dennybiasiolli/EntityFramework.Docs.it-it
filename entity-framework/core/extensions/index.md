# <a name="ef-core-tools--extensions"></a>Strumenti ed estensioni di EF Core

Gli strumenti e le estensioni offrono funzionalità aggiuntive per Entity Framework Core.

Le estensioni vengono compilate da diverse origini. Non tutte le estensioni vengono mantenute nell'ambito del progetto di Entity Framework Core. Quando si prende in considerazione un'estensione di terze parti, valutarne con cura gli aspetti relativi a qualità, licenze, supporto e così via, per essere certi che soddisfi i propri requisiti.
